1-	Baixar o código através do repositório
https://github.com/rinaldo-peligrineli/quatainvestimentos

2-	Após baixar o projeto descompacta-lo executar os seguintes comandos
  2.1 Criar o banco de dados  “teste_quata
  2.2 Php artisan migrate
  2.3 Php artisan db:seed

3 Na pasta do projeto iniciar o servidor do laravel atráves do comando 
  Php artisan serve

4 Iniciar a pagina de filtro através do link  http:localhost:8000/usuarios/index

Obs: o arquivo dados.txt é gerado na pasta public/usuários/yyyy-dd/id_usuario
